Document
-> 커뮤니티 내용에서 제목, 아이디, 글 내용이 필요
-> 커뮤니티 게시글 목록에서 게시글 번호, 제목, 작성자, 작성일, 조회횟수 필요
-> 이름 혹은 제목을 검색 하는 것이 필요
: search=name&search-text=작성일

-> 리뷰에 대한 내용만 가져올 수 있도록 하는게 필요함. (Star가 null이 아닌 것만 가져오도록 하면 됨.)


Product
-> 검색했을때 검색 결과가 도출 될 수 있도록 하는게 필요함
-> 낮은가격, 높은 가격으로 상품을 정렬할 수 있도록(같은 가격이면 정렬 순으로) 처음에는 그냥 정렬 순

User
-> 로그인 시 DB에 존재하는 아이디와 비밀번호와 일치하는지 확인할 수 있도록
-> 회원 가입시 같은 아이디가 존재하는지 확인할 수 있도록
